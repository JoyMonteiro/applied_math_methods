Applied Math Methods (ECS3134)

Vector Spaces

Interpretive Framework

  • Scalars and Vectors are measurements
  • Describe and summarise a set of measurements
  • We will mostly use $\mathbb{R}^n$

Case of One Vector

  • Dot Product and Length
  • Linear Independence
  • Span

Case of $N$ Vectors

  • Dot Product and Length
  • Span
  • Linear Independence
  • Basis and Dimension
  • Suitable Basis?

Summary

  • How to represent and summarise a set of vectors/measurements
  • Dot Product: Similarity, Representativeness
  • Basis $B$: set of Linearly independent vectors (suitable basis?)
  • Span: Space of all vectors that can be expressed as $\Sigma_i\alpha_i b_i$
  • Dimension: Number of vectors in $B$, "Information Content"
